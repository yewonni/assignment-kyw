## 과제 개요
주요 기능:
- 대시보드: 거래 요약, 결제 상태, 거래 트렌드, 가맹점 상태 등 시각화
- 거래 내역 페이지: 필터, 검색, 페이징
- 가맹점 목록 페이지: 검색, 페이징, 상세 모달 확인

---

## 디자인 의도 및 UI/UX 포인트
- **깔끔하고 직관적인 레이아웃**: 각 카드와 테이블을 flex/grid로 배치하여 시각적 정보 우선순위를 명확히 구성
- **색상 활용**: 상태 구분과 데이터 시각화를 위해 직관적인 컬러 팔레트 및 그라데이션 사용 (예: 성공-보라, 실패-빨강, 대기-파랑, 취소-주황)

---

## 설치 및 실행 방법

1. npm install
2. npm run dev


## 환경변수 설정
   프로젝트 실행 전에 `.env` 파일을 프로젝트 루트에 생성하고 아래와 같이 설정합니다.

   
   ```NEXT_PUBLIC_API_BASE_URL="올페이즈의 api base url 입력"```
